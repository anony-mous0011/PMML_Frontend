# Audio Files Required

Place the following audio files in this folder:

| File Name | Person | Description |
|-----------|--------|-------------|
| `modi.mp3` | Modi Ji (Bhagwan) | Modi-Modi-Modi audio clip |
| `tmp8lj.mp3` | Rahul Bacha | Rahul audio clip |
| `rona-band-kijiye.mp3` | Director & Joint Director | "Rona Band Kijiye" audio |
| `sad-meow.mp3` | Me (Bhavi PM) | Sad meow song |

These files are played on hover over each person's name in the inauguration picker.
Supported formats: .mp3, .ogg, .wav
